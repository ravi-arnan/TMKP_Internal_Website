# AI Context - TMKP Internal Website

## Website Purpose

Website ini digunakan untuk **administrasi internal HMI TMKP**: mengelola data anggota, proses verifikasi, laporan keuangan, serta pendataan anggota melalui halaman publik.

## Workflow

1. Pengunjung membuka halaman publik untuk melihat informasi dan mengisi formulir pendataan anggota.
2. Data yang dikirim masuk ke sistem dengan status menunggu validasi.
3. Admin login ke dashboard untuk memeriksa, memverifikasi, atau memperbarui data anggota.
4. Data anggota yang sudah valid digunakan pada modul organisasi, laporan, dan kebutuhan administrasi internal lainnya.
